# CatMash

Technologies utilisés:

* net core 3.1
* Swagger open api
* angular 10
* PrimeNG 10
* docker (pour le deploiement sur le serveur heroku)

* SqLite comme base de données
* Entity Framework core 3
* Serilog
* AutoMapper
* Fluent Validation
* Xunit + NFluent + NSubstitute pour les tes unitaires
